End-to-end React framework with components, layouts, themes, routing, and a generated llms.txt. Cross-platform: web, desktop, mobile, extensions.

npm ↗ GitHub ↗
Install
npm install @bloomneo/uikit
Setup
// 1. Import styles once at app entry
import '@bloomneo/uikit/styles';

// 2. Wrap your app
import { ThemeProvider, ToastProvider, ConfirmProvider } from '@bloomneo/uikit';

<ThemeProvider theme="base" mode="light">
  <ToastProvider />
  <ConfirmProvider>
    <App />
  </ConfirmProvider>
</ThemeProvider>

// 3. Add FOUC prevention to index.html <head> (prevents flash before React mounts)
import { foucScript } from '@bloomneo/uikit/fouc';
<script dangerouslySetInnerHTML={{ __html: foucScript() }} />
One import path

There is exactly one canonical import — always use the flat import:

import { Button, DataTable, Modal, useTheme } from '@bloomneo/uikit';

Deep imports like @bloomneo/uikit/button exist for tree-shaking but are not the canonical form.

Themes

Five built-in themes using OKLCH color science for perceptually uniform palettes.

base

Clean neutral foundation. Good default for any app.

elegant

Refined, subdued palette. Professional and polished.

metro

High contrast, bold. Inspired by transit design systems.

studio

Creative, warm. For design tools and media apps.

vivid

Saturated, energetic. For consumer and marketing apps.

const { theme, setTheme, mode, toggleMode } = useTheme();

setTheme('elegant');   // switch theme
toggleMode();          // light ↔ dark
Component examples
import { Button, Input, Card, CardHeader, CardTitle, CardContent,
         Alert, AlertDescription, Badge, DataTable, Modal,
         Select, Combobox, DatePicker, useToast, useConfirm
       } from '@bloomneo/uikit';

// Button
<Button variant="default">Save</Button>
<Button variant="outline" size="sm">Cancel</Button>
<Button variant="destructive">Delete</Button>

// Card
<Card>
  <CardHeader>
    <CardTitle>Users</CardTitle>
  </CardHeader>
  <CardContent>...</CardContent>
</Card>

// Toast
const { toast } = useToast();
toast({ title: 'Saved', description: 'Changes saved successfully.' });

// Confirm dialog
const confirm = useConfirm();
const ok = await confirm({ title: 'Delete user?', description: 'This cannot be undone.' });
if (ok) { /* proceed */ }
Layout components

  • FooterPage footer with configurable columns

  • PageLayoutFull-page layout wrapper with header/sidebar/footer slots
  • Section / ContainerContent containers with spacing variants
Data components
  • DataTableSortable, filterable, paginated table with column definitions
  • DataListKey-value display component
  • StatCardMetric card with label, value, and trend
Form components
  • Input / Textarea / SelectStandard form inputs
  • ComboboxSearchable select with async option loading
  • DatePicker / DateRangePickerCalendar-based date selection
  • Checkbox / Radio / SwitchBoolean and choice inputs
  • FileUploadDrag-and-drop file input with preview
API hooks
import { useApi, useTheme } from '@bloomneo/uikit';

// API hook — manages loading, data, error state
const api = useApi({ baseURL: 'http://localhost:3000' });
await api.get('/api/users');
// api.data, api.loading, api.error

// Theme hook
const { theme, mode, setTheme, toggleMode, availableThemes } = useTheme();
Cross-platform

Web

Vite + React. Full component library with all themes.

Desktop

Mobile

Capacitor + React. Touch-optimised with scroll containers.

Extensions

Browser extension popup. Constrained layout variants.
